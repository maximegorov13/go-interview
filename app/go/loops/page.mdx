# Циклы

Циклы — важная часть управления потоками выполнения в Go. В языке существует только один тип цикла — `for`, что делает синтаксис минималистичным и универсальным. Его можно использовать для реализации всех традиционных видов циклов, таких как `while`, `do-while` и `for` в других языках.

## Типы циклов в Go

1. **Обычный цикл с условием:**

   `for инициализация; условие; изменение` — стандартный цикл `for`, аналогичный C-like языкам.

2. **Цикл с единственным условием:**

   Упрощенная форма `for`, аналогичная `while` в других языках. Условие проверяется перед каждой итерацией.

3. **Бесконечный цикл:**

   Отсутствие всех параметров создает бесконечный цикл.

4. **Итерация с `range`:**

   Используется для обхода срезов, массивов, карт, строк и каналов.

## Почему в Go нет `while` и `do-while`?

Go отказался от отдельных циклов `while` и `do-while`, заменив их универсальным `for`. Это решение было принято для уменьшения сложности языка и обеспечения легкости чтения кода.

## Особенности работы с `range`

- **Массивы и срезы:** Возвращаются индекс и копия значения элемента.
- **Карты:** Возвращаются ключ и значение пары.
- **Строки:** Возвращаются индекс байта и символ (руна).
- **Каналы:** Возвращается только значение, так как у каналов нет индексов.

## Почему `range` возвращает копии элементов?

Это решение обеспечивает безопасность данных. Возвращаемая копия элемента предотвращает случайные изменения исходных данных. Для изменения значений необходимо использовать индекс элемента.

## Использование меток в циклах

Метки позволяют управлять вложенными циклами. С помощью `break` или `continue` можно завершить внешний цикл из внутреннего, что делает код более читаемым в сложных структурах. Метки определяются перед циклом и вызываются по имени.

## Особенности обработки строк в `range`

При итерации по строке `range` работает с рунами (`rune`), что позволяет правильно обрабатывать символы Unicode, занимающие несколько байтов. Индекс указывает на начальный байт символа в строке, а не на сам символ.

## Ключевые вопросы

- **Универсальность `for`:** Замена `while` и `do-while` упрощает синтаксис.
- **Обязательные фигурные скобки:** Нет круглых скобок вокруг условий.
- **Работа с `range`:** Возвращает копии данных, поддерживает разные типы коллекций.
- **Метки:** Обеспечивают управление вложенными циклами.
- **Область видимости переменных:** Переменные, объявленные в `for`, видны только внутри цикла.
- **Поддержка UTF-8:** Корректная работа с многоязычными строками через `range`.
