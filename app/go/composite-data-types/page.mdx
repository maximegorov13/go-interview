# Составные типы данных

| **Тип**            | **Синтаксис** | **Описание**                            | **Пример**                                    |
| ------------------ | ------------- | --------------------------------------- | --------------------------------------------- |
| **Массивы**        | `[n]T`        | Последовательность фиксированной длины. | `[5]int{1, 2, 3, 4, 5}`                       |
| **Срезы (слайсы)** | `[]T`         | Динамический массив.                    | `a := []int{1, 2, 3}`                         |
| **Карты (мапы)**   | `map[K]V`     | Ассоциативные массивы (словарь).        | `m := map[string]int{"key": 1}`               |
| **Структуры**      | `struct{}`    | Пользовательские типы с полями.         | `type Person struct { Name string; Age int }` |
| **Каналы**         | `chan T`      | Передача данных между горутинами.       | `c := make(chan int)`                         |
| **Функции**        | `func(T) R`   | Функции как значения.                   | `func add(a, b int) int { return a + b }`     |
