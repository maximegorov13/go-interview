# Типы данных

| **Категория**                     | **Тип**                | **Описание**                                   | **Диапазон / Значения**                                 |
| --------------------------------- | ---------------------- | ---------------------------------------------- | ------------------------------------------------------- |
| **Числовые (целые)**              | `int`                  | Целое число (зависит от архитектуры).          | 32/64-бит (в зависимости от ОС).                        |
|                                   | `int8`                 | 8-битное целое число.                          | -128 до 127                                             |
|                                   | `int16`                | 16-битное целое число.                         | -32,768 до 32,767                                       |
|                                   | `int32`                | 32-битное целое число.                         | -2,147,483,648 до 2,147,483,647                         |
|                                   | `int64`                | 64-битное целое число.                         | -9,223,372,036,854,775,808 до 9,223,372,036,854,775,807 |
|                                   | `uint`                 | Беззнаковое целое (архитектурно-зависимо).     | 0 до 4,294,967,295 (на 32-битных ОС)                    |
|                                   | `uint8` (байт)         | 8-битное беззнаковое число (байт).             | 0 до 255                                                |
|                                   | `uint16`               | 16-битное беззнаковое число.                   | 0 до 65,535                                             |
|                                   | `uint32`               | 32-битное беззнаковое число.                   | 0 до 4,294,967,295                                      |
|                                   | `uint64`               | 64-битное беззнаковое число.                   | 0 до 18,446,744,073,709,551,615                         |
| **Числовые (с плавающей точкой)** | `float32`              | Число с плавающей точкой (одинарная точность). | ±1.18e-38 до ±3.4e38                                    |
|                                   | `float64`              | Число с плавающей точкой (двойная точность).   | ±2.23e-308 до ±1.8e308                                  |
| **Комплексные числа**             | `complex64`            | Комплексное число с `float32`.                 | Реальная и мнимая части - `float32`                     |
|                                   | `complex128`           | Комплексное число с `float64`.                 | Реальная и мнимая части - `float64`                     |
| **Логические**                    | `bool`                 | Логическое значение `true`/`false`.            | `true`, `false`                                         |
| **Строковые**                     | `string`               | Последовательность символов UTF-8.             | Строки любой длины                                      |
| **Руны (символы)**                | `rune` (алиас `int32`) | Символ Unicode.                                | UTF-8 код символа                                       |
| **Указатели**                     | `*T`                   | Указатель на переменную типа `T`.              | `nil` или адрес переменной                              |
| **Интерфейсы**                    | `interface{}`          | Универсальный тип для любых значений.          | Любой тип                                               |
